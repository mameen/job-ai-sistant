# Career KB — private layer

**Local only — never commit.** Lives at `.kb/private/`.

Sensitive preferences, comp targets, flags, reflections, and application history.
Job Researcher and Application Coach may read for fit and strategy; never expose
in public artifacts without your intent.

## Files

| File / folder | Purpose |
|---|---|
| `career_goals.md` | Target roles, level, industry |
| `compensation_targets.md` | Salary, equity, benefits |
| `location_preferences.md` | Locations, remote/hybrid |
| `red_flags.md` | Hard no |
| `yellow_flags.md` | Caution signals |
| `performance_themes.md` | Review themes (paraphrased) |
| `observations/` | Evidence-backed learnings |
| `application_history/` | Past applications |
| `reflections/` | Personal notes |

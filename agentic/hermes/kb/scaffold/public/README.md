# Career KB — public layer

**Local only — never commit.** Lives at `.kb/public/`.

Application-safe facts: employment, projects, skills, STAR stories suitable for
resumes and cover letters. The KB Manager curates; Job Researcher and Application
Coach read (read-only).

## Files

| File / folder | Purpose |
|---|---|
| `master_resume.md` | Canonical resume source |
| `skills.md` | Skills mapped to experience |
| `projects/` | One file per major project |
| `star_stories/` | Behavioral interview stories |
| `education.md` | Degrees, coursework |
| `certifications.md` | Certs and dates |

Edit in place. Proposals from Application Coach merge here only after your approval.
